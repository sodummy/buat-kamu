# Folder Lagu (`audio/`)

Taruh file lagu kamu di folder ini (contoh: `lagu-kita.mp3`).

Terus, di `script.js` isi bagian `src` jadi misalnya:

- `src: 'audio/lagu-kita.mp3'` (buat halaman isi / opening)
- `src: 'audio/lagu-terakhir.mp3'` (buat halaman surat)

Catatan:
- Paling aman pakai format `.mp3`.
- Nama file sebaiknya tanpa spasi (pakai `-` atau `_`).
